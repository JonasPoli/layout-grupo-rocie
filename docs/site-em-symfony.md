# Como fazer
Você deve criar um novo projeto em symfony.
veja como foi montado o projeto /Users/jonaspoli/work/html/layouts/cetec-site-2026 e faça o mesmo (similar) em layout-gruporocie
Coloque o symfony 7, tailwindcss, shoelace, stimulusjs, tinymce e tudo mais que existe no projeto cetec-site-2026

Você deve criar toda a estrutura necessária para o banco de dados e deve criar um command para injetar dados, similar ao cetec-site-2026/src/Command/AppGenerateDummyDataCommand.php mas levando em consideração que é um site de produtos:
